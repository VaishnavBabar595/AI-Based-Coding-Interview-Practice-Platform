## 📖 Project Overview

AI Coding Interview Practice Platform is a full-stack coding practice website where users can solve programming problems and receive AI-generated feedback on their solutions.

The platform provides an interactive coding environment similar to modern coding practice platforms. Users can write code in an online editor, submit solutions, and receive suggestions for improvement.

This project demonstrates concepts of full-stack development, REST APIs, database management, and AI-based code analysis.

---

## 🚀 Key Features

- 💻 Online coding environment with code editor
- 🧠 AI feedback on submitted solutions
- 📊 Track coding problem submissions
- 🗂 Problem management system
- 💾 Store submissions using database
- ⚡ Full-stack MERN architecture

## 💻 How to Run Locally

### 1️⃣ Clone the repository

bash
git clone https://github.com/YOUR_USERNAME/ai-coding-interview-platform.git

### 2️⃣ Navigate to the project folder

cd ai-coding-interview-platform

### 3️⃣ Run Backend

cd server
npm install
node index.js


### Backend runs on:

http://localhost:5000

### 4️⃣ Run Frontend

Open a new terminal:

cd client
npm install
npm run dev

### Frontend runs on:

http://localhost:5173
